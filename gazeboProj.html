Gazebo Projects

Gazebo tutorials series:
Gazebo has a bunch of tutorials intended to teach you how to navigate the simulation environment and how to use the UI. Here is the link to the tutorial docs.

Building a simulated robot:
Simulated robots and worlds are described with .sdf(Simulation Description Format) files. For the moment, a simple robot with 3 wheels is ok. We can read the .sdf files via gz sim FILE_NAME.sdf
We can describe the robot manually via these .sdf files, but using already existing markets and modifying the files seems to be the way.

Moving the robot:
We can move the robot by listening to ros2 topics, or in this case we move the robot by listening to the topic where keyboard presses can be interpreted.
gz topic -e -t /keyboard/keypress

Sensors:
We can communicate to the simulated environment using sensors. Sensors can be used via plugins and they publish the necessary information to certain topics that can be subscribed to. This is where to ROS2 connection becomes very clear.
