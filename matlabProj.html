Matlab robotics sim tutorial:


Robotics is hard. An 'easy' way to get into robotics and control is using Matlab and its robotics tool box. Here is the link to the tutorial that I followed.









Building a robot model:
We can build robot models in Matlab using Simscape. We describe each joint, link and transformation matrices of the link origin. After mapping some signals as actuators we can get a RRR robot that can be moved via the input signals.

Trajectory planning:
If we cleverly adjust the input signals using inverse-dynamics, forward-dynamics we can plan the trajectory of the End-Effector as we want.
