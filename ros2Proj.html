ROS2 tutorials

Learning ROS2 using the ROS2 online tutorial.

How ros2 communicates:
ROS2 has certain types of communication that parts called nodes use to communicate to each other. These ways are via topics, actions and services.

Using 'turtlesim' package to simulate node interactions:
We can move the robot by listening to ros2 topics, or in this case we move the robot by listening to the topic where keyboard presses can be interpreted.
ros2 run turtlesim turtlesim_node
